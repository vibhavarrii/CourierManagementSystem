# Courier Management System (CMS)

> **"Efficient Delivery at Your Doorstep"**

A full-stack web application that streamlines courier operations with complete CRUD functionality.

---

## Overview

The CMS allows users to place couriers, track delivery status, submit and delete feedback, manage staff registrations, and handle franchise additions and deletions — all through a clean, user-friendly interface.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, Bootstrap |
| Backend | PHP |
| Database | MySQL |

---

## Tools Used

- Visual Studio Code
- XAMPP
- Git & GitHub

---

## Features

- **User Auth** — Signup and login to access core features
- **Place a Courier** — Registered users can submit courier requests
- **Delivery Tracking** — Check delivery status (updated after 24 hours)
- **Feedback System** — Submit or delete your feedback
- **Staff Management** — Register and update staff details
- **Franchise Management** — Add or remove franchise branches

---

## Getting Started

1. Clone the repository and install XAMPP
2. Start Apache and MySQL modules
3. Create a database named `cms` and set up the required tables
4. Move project files into the `htdocs` folder
5. Update DB credentials in `partials/_dbconnect.php`
6. Navigate to `http://localhost/login/signup.php` to register and get started
